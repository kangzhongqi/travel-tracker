# 旅行打卡地图

## 功能
- 🚇 **北京地铁打卡**：27 条线路、410 个站点，点击打卡标记
- 🇨🇳 **中国旅行足迹**：地级行政区打卡，省份统计

## 使用
打开 [https://YOUR_USERNAME.github.io/travel-tracker/](https://YOUR_USERNAME.github.io/travel-tracker/)

数据存储在浏览器本地，不会上传到任何服务器。

- `北京地铁打卡地图.html` — 桌面版

- `中国旅行足迹地图.html` — 桌面版
- `中国旅行足迹-iPhone.html` — iPhone 版
